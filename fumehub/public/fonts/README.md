# Dublin Font Files

Please add your Dublin font files here:

- Dublin-Regular.woff2 (or .woff, .ttf)
- Dublin-Bold.woff2 (or .woff, .ttf)

You can download Dublin font from:
- Adobe Fonts: https://fonts.adobe.com/fonts/dublin
- Or purchase from a font foundry

Once you add the font files, the website will automatically use Dublin font throughout.
